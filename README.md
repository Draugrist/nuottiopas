# Nuottiopas

Yksinkertainen musiikkinuottien harjoittelusovellus.

https://nuottiopas.draugrist.fi/

## Asennus

```bash
npm install
```

## Kehitystila

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Projektin rakenne

- `src/` sovelluksen lähdekoodi
- `public/` staattiset tiedostot
- `index.html` sovelluksen pääsivu
